Вузли можна вибирати за допомогою випадаючого списку або кліком по карті.
Прибрати вузол можна за допомогою кліку по маркеру.

За допомогою сервера Node.js є можливість додавати нові вершини.
Для цього у файлі server.js розкоментуйте рядок з функцією addNode, запустіть файл server.js та в браузері перейдіть за посиланням http://127.0.0.1:3000/.

Параметри функції addNode: addNode(назва вузла, широта, довгота, масив із назв зв'язаних станцій)
